## Hallazgos principales

### 5 Modelos de monetización para AI agents

**1. Usage-Based Pricing**
- Cobra por tokens, API calls, tiempo de ejecución, o frecuencia de acciones
- El cliente paga según consume
- Ejemplo: OpenAI cobra por token

**2. Subscription Licensing**
- Precio fijo mensual/anual basado en usuarios, agentes, o features
- Ideal para revenue predecible
- Ejemplo: $99/mes por agente

**3. Outcome-Based Pricing**
- Cobra por resultado medible — leads generados, tickets resueltos, horas ahorradas
- El más alineado con ROI del cliente
- Intercom pricing Fin: $0.99 por resultado

**4. Bundled SaaS Pricing**
- Agente como feature dentro de producto SaaS existente
- Monetización indirecta pero poderosa
- Ejemplo: CRM con AI agent incluido

**5. Marketplace/API Monetization**
- Venta via GPT Store, Zapier, Salesforce
- Cobra por integración o ejecución
- Ejemplo: vender en Apollo.io

### Costos de inference bajaron 280x

Desde noviembre 2022 a octubre 2024, el costo de inference a nivel GPT-3.5 cayó 280 veces. Esto hace viable el pricing por micro-transacciones.

### Para BRAIN — modelos más viables

1. **Subscription + Usage** (modelo híbrido)
   - $30-100/mes base + $0.10-0.50 por ejecución
   - Incluye seat para Dani como admin

2. **Outcome-based** para servicios específicos
   - Research como servicio: $500-2000/proyecto
   - Automation setup: $200-500/hora
   - Coding agent service: $100-300/hora

3. **Bundled** con servicios existentes
   - Si Dani ya ofrece servicios de automation, añadir BRAIN como differentiator

El mercado está creciendo 46-49% anual. Los modelos más exitosos combinan subscription con usage, y el outcome-based está ganando tracción para empresas. BRAIN puede implementar cualquiera de estos modelos dependiendo del cliente target.
