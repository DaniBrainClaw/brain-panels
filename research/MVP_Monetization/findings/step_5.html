Hallazgos principales

Las opciones para solopreneurs

| Platform | Fees | Best for | |----------|------|----------| | Stripe | 2.9% + €0.20 | Tech-savvy, control | | PayPal | 3.49% + €0.20 | Simple, familiar | | Paddle (MoR) | 5% | Tax handling, compliance | | Lemon Squeezy (MoR) | 5% | Digital products, creators |

Stripe vs PayPal

Stripe:
  • ✅ API documentation unmatched
  • ✅ Developer-friendly
  • ✅ Control total
  • ❌ Requires tech setup
  • ❌ Tú manejas taxes
  • ❌ 2.9% + €0.20 por transaction
PayPal:
  • ✅ Rápido de setup
  • ✅ Familiar para clientes
  • ✅ No requiere código
  • ❌ Fees más altos (3.49%)
  • ❌ Menos control
  • ❌ Menos professional
Recommendation para Dani: Stripe + PayPal ambos

El modelo Merchant of Record (MoR) — Paddle/LemonSqueezy

Por qué importa:
  • Con Stripe: 100 clientes = 1,200 invoices + 1,200 tax reports al año
  • Con MoR: 100 clientes = 12 payouts (no invoices, no tax reports)
  • El MoR gestiona todo: taxes, VAT, customer support, refunds
Costo extra: 5% vs 2.9% — pero ahorra tiempo enorme

Pricing models para AI services

Modelo 1: Flat subscription (recomendado para Dani)
  • €500/mes — 30 tasks
  • €1,500/mes — 100 tasks + setup
  • €3,000/mes — unlimited
Modelo 2: Hybrid (base + overage)
  • €500 base + €25/task adicional
  • Protege margins
Modelo 3: Per-project
  • €300-500 por proyecto
  • Para workflows específicos
Modelo 4: Usage-based (evitar al inicio)
  • €0.50-2 por task
  • Dificultad para predecir revenue

Setup para Stripe (lo que necesitas)

Step 1: Account business
  • Stripe account: stripe.com
  • Necesitas: CIF/NIF, cuenta bancaria
  • Verification: 1-2 días
Step 2: Payment link (sin código)
  • Crea payment links desde dashboard
  • Ejemplo: "Paga tu AI Automation Setup"
  • Copia y pega en email, WhatsApp
Step 3: Integra en landing page
  • Botón: "Pagar €500"
  • Link directo a Stripe payment
  • No necesitas web developer
Step 4: Para subscriptions
  • Dashboard → Billing → Products
  • Crea 3 products (tiers)
  • Activa recurring

Setup para PayPal

Option 1: Payment links
  • paypal.com/business/payments
  • Crea link: paypal.me/tunegocio/500
  • Envía por email/WhatsApp
Option 2: Facturas
  • invoicing.paypal.com
  • Crea factura, envía
  • Cliente paga con PayPal o tarjeta

Cómo cobrar sin website complejo

El flujo simple:
  • Landing page con tu oferta
  • Botón "Book Call" o "Pagar"
  • En el call, si dice yes → envías payment link
  • Pagado → start project
Herramientas:
  • Carrd.co (landing page gratis)
  • Stripe payment links
  • PayPal.me
  • WhatsApp per发送 link
