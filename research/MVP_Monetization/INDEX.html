Los 7 Pasos para €1,000/mes

Paso 1: Define tu servicio

"AI Automation Setup para [nich]"
  • Nich: coaches, consultants, small businesses
  • Servicio: 1-3 workflows automatizados
  • Precio: €500 setup + €200/mes maintenance

Paso 2: Crea tu landing page

  • 1 página con: Problem → Solution → Pricing → CTA
  • Herramienta: Carrd.co (gratis)
  • Tiempo: 2 horas
  • BRAIN genera el copy

Paso 3: Consigue tu primer cliente

  • Comment outreach en Reddit/Facebook Groups
  • Plantilla: "Vi que tienes [pain] — construí algo que lo resuelve"
  • Framework: LRP (Listen, Repeat, Poke)
  • Precio primer cliente: €300-500 (bajo para testimonial)

Paso 4: Onboarding rápido

  • Google Form (8 preguntas)
  • Email template automatización
  • Loom walkthrough (graba uno, usa para todos)

Paso 5: Payment setup

  • Stripe: stripe.com (2.9% + €0.20)
  • PayPal: paypal.me/tunombre
  • Payment links para cada tier

Paso 6: Contract simple

  • Template básico con: scope, payment, timeline, IP, termination
  • Guarda como PDF
  • Firme antes de empezar

Paso 7: Automatiza delivery

  • Zapier para workflows repetitivos
  • BRAIN como project manager
  • Templates para todo

Pricing Tiers Recomendado

| Tier | Precio | Incluye | |------|--------|---------| | Starter | €500 | 1 workflow, 30 días soporte | | Professional | €1,500 | 3 workflows, setup, 60 días soporte | | Elite | €3,000 | Unlimited workflows, priority, monthly call |

El Error Fatal que Evitar

No empezar por:
  • Landing page perfecta
  • Brand complejo
  • Múltiples servicios
  • Esperar a tener todo
Sí empezar por:
  • 1 servicio específico
  • 1 cliente real
  • 1 workflow que functions
  • 1 testimonial
