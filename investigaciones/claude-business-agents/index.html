Prioridad ALTA

Guía para reducir costes 40-70%

Reglas clave

  • Nueva conversación cada 15-20 mensajes — Message 30 puede costar 50,000+ tokens
  • Modelo correcto por tarea: Haiku ($1/M) vs Sonnet ($3/M) vs Opus ($15/M)
  • Phased approach: Architecture (Opus) → Core (Sonnet) → Module (Sonnet, new chat) → Review (Opus)
  • Desactivar tools no usadas — cada MCP carga en cada mensaje

Regla #1 más importante

Claude re-envía toda la conversación cada mensaje. Message 1 = 500 tokens. Message 30 = 50,000+ tokens.
